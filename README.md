# ÖBA otomatik oynatım
Sizin için videoyu otomatik olarak başlatır ve bittikten sonra bir sonraki videoya geçer
# Kurulum
## Chrome (ve türevleri)
Bir "program" yöneticisi gereklidir, [Violentmonkey](https://chrome.google.com/webstore/detail/violent-monkey/jinjaccalgkegednnccohejagnlnfdag) yada [Tampermonkey](https://tampermonkey.net/) önerimdir. Bunlardan birini kurduktan sonra [buraya tıklayarak kurulumu yapabilirsiniz](https://github.com/yiitbrak/OBA_autoplay/blob/main/oba_ap.user.js)
## Firefox
[Violentmonkey](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/) veya [Tampermonkey](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/) gereklidir. Bunlardan birini kurduktan sonra [buraya tıklayarak kurulumu yapabilirsiniz](https://github.com/yiitbrak/OBA_autoplay/blob/main/oba_ap.user.js)
